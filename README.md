# LocateOffline

An offline-first spatial data engine designed to seed, sync, and query spatial vector features entirely local to a target device using the standard Open Geospatial Consortium (OGC) GeoPackage format built on SQLite.

## Prerequisites
- Python 3.11+
- SQLite3 with R-Tree module enabled (Default on most modern Linux environments)

## Installation
```bash
git clone [https://github.com/MichaelMukiibi/locate-offline.git](https://github.com/MichaelMukiibi/locate-offline.git)
cd locate-offline
uv pip install -r requirements.txt

```
